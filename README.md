本实验系统基于TwinCAT 3搭建实时控制平台，开发以Beckhoff控制器为核心的硬件控制系统和上位机软件系统，以新松DUCO GCR12-1300协作机器人为对象，实验设备如图6所示，协作机器人坐标轴定义如图7所示，其MDH参数如表2所示。硬件系统包括六自由度协作机器人、Beckhoff控制器、I/O 模块及 EtherCAT 高速总线网络等，可通过EtherCAT总线协议，以4 ms的周期实现控制器与机器人伺服驱动器之间的高精度同步。上位机软件系统基于 TwinCAT 3开发，实现了机器人关节角度、角速度、角加速度及伺服电机驱动电流的采集与实时监控。该系统能有效满足实验对高精度数据采集与控制的需求。
